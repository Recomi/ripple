# Ripple.js
鼠标点击水波纹插件

### 用法
引入js

```html
<script src="ripple.js"></script>
```
在需要添加效果的元素上加上： data-ripple="ripple"

调用
```javascript
<script type="text/javascript">
    new Ripple()
</script>
```

```html
<div data-ripple="ripple">
    div标签
</div>
```
